工作人員操作手冊 / Staff Manual

Phone consult operation guide for staff
無權限

1. 角色說明（Role Overview）

  • Owner：管理全部
    Owner: Full access
  • Booking：管理時段、處理改期
    Booking: Manage slots and reschedules
  • Fulfillment：不可操作電話諮詢
    Fulfillment: No access to phone consult

2. Phone Consult 流程圖（文字版）

  1. Owner 建立 slot
  2. 使用者預約 → hold
  3. 成立訂單
  4. 通話完成
  5. 若需改期 → 申請 → 管理員處理
Owner creates slots → user holds a slot → order created → call completed → reschedule request (if needed) → admin handles.

3. 時段管理規則

  • 一次只能 hold 一個 slot
  • Slot 有效時間（系統控制）
  • 過期自動釋放
  • 不可手動修改訂單 slot
Only one hold per user. Hold TTL is system-controlled and expires automatically. Do not manually edit order slots.

4. 改期處理規則

  • 僅限 48 小時前
  • 一筆訂單只能申請一次
  • 可 Approve / Reject
  • 會寄雙語通知信
Requests must be 48+ hours before the slot, one request per order. Approve or reject; bilingual emails are sent automatically.

5. 禁止事項（紅字）

  • 不可私下改 slot
  • 不可代客人建立訂單
  • 不可刪除歷史訂單
  • 不可修改付款資料
Do not modify slots privately, create orders on behalf of customers, delete order history, or edit payment data.

6. 常見問題（FAQ）

為什麼 slot 不能選？
可能未發布、已滿、或已被保留。
為什麼不能改期？
已超過 48 小時限制或已送出改期申請。
為什麼我看不到電話算命？
非 admin/booking 或目前無可用時段。
FAQ: Slots may be unpublished, full, or held. Reschedule may be too late or already requested. Visibility depends on role and availability.
